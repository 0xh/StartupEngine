Playing with events and the TColor object

This example listens to the colorpickerChange and colorpickerCreate events and changes the background color of the parent element using the complementary color of the selected one.
This also shows how to use the TColor object, which extends tinycolor.
In all events the e event variable contains the colorpicker and color objects, but in the colorpickerChange and colorpickerInvalid events it also contains value, representing the color string.

Code
<div id="cp1" data-color="rgba(194, 39, 219, 0.4)"></div>
<script>
  $(function () {
    $('#cp1')
        .colorpicker({
          inline: true,
          container: true
        })
        .on('colorpickerChange colorpickerCreate', function (e) {
          e.colorpicker.picker.parents('.card').find('.card-header')
              .css('background-color', e.color.toRgbString());

          e.colorpicker.picker.parents('.card-body')
              .css('background-color', e.color.getScheme('complementary')[0].string('rgb'));
        });
  });
</script>
